# Support

Thank you for using the Alibaba Cloud Agent Toolkit scaffold.

## Documentation

- Repository documentation currently lives in [README.md](README.md) and the plugin or rules directories.

## Bug Reports

Open a GitHub issue with:

- The plugin or directory involved
- Your AI assistant and version
- Your operating system
- Steps to reproduce

## Feature Requests

Open a GitHub issue with a clear description of the requested capability and the Alibaba Cloud workflow it should support.

## Security Vulnerabilities

Do not open a public issue for security concerns. Follow [SECURITY.md](SECURITY.md).
